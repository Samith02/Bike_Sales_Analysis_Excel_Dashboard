# Bike Sales Analysis Dashboard

## Project Overview
This project analyzes customer bike purchase behavior using Microsoft Excel. The dashboard provides insights into customer demographics, income levels, commute distance, age groups, and purchasing trends.

## Objectives
- Analyze factors affecting bike purchases
- Identify customer demographics and income trends
- Create an interactive dashboard using Excel
- Generate business insights from raw sales data

## Tools & Features Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Conditional Formatting
- Interactive Dashboard
- Excel Formulas

## Dataset Information
The dataset contains customer-related information such as:
- Gender
- Age
- Income
- Occupation
- Marital Status
- Region
- Commute Distance
- Bike Purchase Status

## Dashboard Insights
Key insights obtained from the dashboard:

- Customers with higher income showed greater bike purchasing probability
- Middle-aged customers contributed significantly to bike purchases
- Commute distance influenced buying behavior
- Regional trends helped identify potential customer segments

## Project Workflow
1. Imported raw dataset
2. Cleaned and organized data
3. Created pivot tables for analysis
4. Built charts and visualizations
5. Designed interactive dashboard using slicers
